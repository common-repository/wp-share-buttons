== Installation ==

### Uploading in WordPress Dashboard

1. First download the ZIP file from Wordpress website
2. Log in to your website administrator panel
3. Go to the 'Add New' in the plugins dashboard, click “Upload Plugin”
4. Upload [WordPress Share Buttons](http://wordpress.org/plugins/wp-share-buttons) ZIP file by choosing it from your computer
5. Click Install Now button
6. Then click Activate Plugin button
7. You can see the plugin installed on Wordpress left menu

### Using The WordPress Plugins Dashboard

1. Go to the 'Add New' in the plugins dashboard
2. Search for 'Share Button Huge-IT'
3. Click Install Now button
4. Then click Activate Plugin button
5. You can see the plugin installed on Wordpress left menu

### Using FTP

1. Download the ZIP file from Wordpress website
2. Extract the share-button directory to your computer
3. Upload the share-button directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
5. You can see the plugin installed on Wordpress left menu

==Share buttons user manual==

### Adding

To Add SPlugin after activation, go start customize them

Step 1 Free Customizations

Social Medias. Choose the social websites, from this list, that you want to display in your post.
Size. Set the size for the Share icons.
Button Position. Select the position of the Share buttons on your webpage.

### 1.1 Pro Customizations

Style. Choose the style from the list

*  Block Has Background. Choose to have background
*  Block Background Color. Select the color for background.
*  Block Border Size. Set the size of the border.
*  Block Border Color. Choose the color of your border.
*  Block Border Radius. Choose the radius for border corners.
*  Margin From Content. Set the margin of your content.


Title

*  Title Text. Write a text
*  Title Position. Choose the position of the text		
*  Title Font Size. Set the font size for the title
*  Title Color. Choose a color for the title
*  Title Font Style(Family). Set Font style for title text


Custom Styles

*  Margin Between Buttons. Set the distance between buttons
*  Has Background. Choose whether to have background for the buttons
*  Background Padding. Choose the padding size for buttons’ background
*  Background Color. Select background color
*  Border Size. Set >0 to have the border and adjust it for buttons
*  Border Style. Choose a style for border of the share button
*  Border Color. select a color for buttons’ border
*  Border Radius. Set radius for buttons’ border corners
*  All changes will be displayed next to the options in demo content
